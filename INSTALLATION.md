# Installationsguide för PWA

## Ta bort och installera om appen (för att få nya ikoner)

### Metod 1: Via Brave-inställningar (Enklast)

1. Öppna Brave på telefonen
2. Gå till: `brave://apps` (skriv detta i adressfältet)
3. Hitta "Task Board" eller "Dynamic Task Board"
4. Klicka på den → "Ta bort" eller "Uninstall"
5. Gå tillbaka till: `https://ama-x.github.com/DynTaskList`
6. Installera på nytt via "Lägg till på startskärmen"

### Metod 2: Ta bort från hemskärmen

1. Långtryck på ikonen på hemskärmen
2. Välj "Ta bort" eller dra den till papperskorgen
3. Öppna Brave och gå till: `https://ama-x.github.com/DynTaskList`
4. Installera på nytt

### Metod 3: Rensa cache i Brave

1. Öppna Brave
2. Gå till: `brave://settings/clearBrowserData`
3. Välj "Cached images and files"
4. Välj "Last hour" eller "Last 24 hours"
5. Klicka "Clear data"
6. Gå tillbaka till: `https://ama-x.github.com/DynTaskList`
7. Installera på nytt

### Metod 4: Rensa via menyn

1. Öppna Brave
2. Klicka på menyn (tre prickar)
3. Gå till "Settings" → "Privacy and security"
4. Klicka på "Clear browsing data"
5. Välj "Cached images and files"
6. Klicka "Clear data"
7. Installera appen på nytt

## Kontrollera att maskable icons laddas

Efter att ha installerat om, testa:
1. Öppna: `https://ama-x.github.com/DynTaskList/manifest.webmanifest` i webbläsaren
2. Kontrollera att `icon-192-maskable.png` och `icon-512-maskable.png` finns i listan
3. Om de inte syns, vänta lite längre (GitHub Pages kan ta några minuter att uppdatera)
